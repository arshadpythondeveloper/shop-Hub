# 🛒 Shop Hub - E-Commerce Web Application

Shop Hub is a full-stack e-commerce web application built using Flask framework. It provides user authentication, product management, and shopping cart functionality with a MySQL database backend.

---

## 🚀 Features

- User Registration & Login System
- Product Listing & Details Page
- Add to Cart / Remove from Cart
- Admin Dashboard (Manage Products)
- Secure Database Integration (MySQL)

---

## 🛠️ Technologies Used

- Python (Flask)
- JavaScript
- HTML5 / CSS3
- MySQL
